    Fifth Generation Firmware 1.9.1

    Released 1.19.2016

    All Fifth Generation MakerBot 3D Printers

    Features
    • Optimizations for Smart Extruder+
    • Your MakerBot 3D Printer will now pre-heat the extruder and start the homing process while MakerBot Desktop is creating your print file.
    • Option to unload filament automatically after a print is complete
    • Better error handling during filament loading should catch cases where a short piece of filament gets stuck below the drive gear


    Fifth Generation Firmware 1.8.0

    Released 11.3.2015

    All Fifth Generation MakerBot 3D Printers

    Features
    • Print flow improvements (“clear build plate” before the slice)
    • Cancelled/completed state for prints
    • Firmware updating from a 5th Generation printer's LCD interface
    • New easy-to-read printer logs
    • Toolhead error reduction (specifically errors 65 and 1032)
    • Print performance profiling and minor performance improvements
    • Print suspension rework (de-accelerate immediately following error)

    MakerBot Replicator Z18

    • Improved build plate leveling process

    Fifth Generation Firmware 1.6

    Released 1/5/15

    • 1.6.3: Bug fixes and improvements including fix for the "internal connection timeout" error during extended prints

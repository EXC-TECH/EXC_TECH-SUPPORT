Installing MakerBot Print



    Download MakerBot Print

    If you are downloading MakerBot Print for the first time, or if you are re-installing MakerBot Print, watch the video below for a step-by-step guide to making sure you have the correct version installed, and are installing properly. 

    Note: The video below outlines installation of the three softwares we have available for a Mac operating system. 


    0:00
    10:59
    0:00


    The current release of MakerBot Print is compatible with the MakerBot Sketch, MakerBot Method, MakerBot Method X, MakerBot Replicator+, and Replicator Z18*, Mini+, and 5th Generation*

    *Must be using a Smart Extruder+ and not a Smart Extruder 

    1. Navigate to www.makerbot.com/print

    2. Click "Download MakerBot Print". Select your operating system from the dropdown menu on the download pack and click Download. Be sure to have the correct system requirements.

    Do You Have a Legacy Device?

    1. Navigate to www.makerbot.com/print

    2. Click "Download MakerBot Print". On the next page, scroll to the bottom of the page and find the header that says "Legacy Printer Support". Under this header there are options to download one of our two legacy softwares. 

    MakerBot Print Legacy 4.3 is compatible with the MakerBot Method*, Method X*, MakerBot Replicator+, Replicator Z18, Mini, Mini+, 5th Generation, and Replicator 2** and 2X.**

    *Some material and Extruder options may be limited 

    **Legacy Print does not support USB Connectivity for these devices and will be Export-Only. 

    MakerBot Desktop is compatible with MakerBot Replicator Z18, Replicator Mini, Mini+, 5th Generation, Replicator 2, Replicator 2X, Replicator Dual, Thing-O-Matic, and Cupcake CNC


    Do You Have Full Adminstrator Permissions?

    To install Print, a user must have administrator permissions. In MacOS, this information is available by accessing Apple > System Preferences > User & Groups. In Windows, you can see your access level through Start > Window Settings > Accounts.

    In an office or school, this means that the system administrator will likely need to perform the installation. Print does NOT allow the user to pick the install location and will install in the user directory of the account active at the time of the installation. **Note - If you are the administrator, do NOT right click on the installer and click Run as administrator, please wait for the installer to prompt you for the Administrator credentials.

    About the Software

    The MakerBot Print software is set up to be installed individually for each user and is not optimized for any installation management platforms. We are unfortunately unable to provide alternate installers to accommodate simultaneous multi-user installations. If key files are moved post-install, the Print software will still default to its intended paths upon its next update. These updates are mandatory, with cancelations only delaying the process by 12 hours. Downloads are indicated by a progress bar towards the top of the virtual build plate on the software. Once the download completes, restarting the application will automatically launch the installer, which will require an administrator to be completed.

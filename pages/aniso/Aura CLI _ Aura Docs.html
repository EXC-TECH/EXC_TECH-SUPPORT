# Aura CLI alfa

La versión alfa de Aura CLI se incluye en Aura 1.24 y posteriores. Aura.CLI.exe se encuentra en el directorio del programa Aura.

# Uso

Procesar código g para el modelo:
aura.cli --center --models part.stl --output part.gcode --session settings.aus
Procesar código g para varios modelos:
aura.cli --models part.stl part2.stp --output part.gcode --session settings.aus
Procesar código g con reglas de disposición personalizadas para piezas:
aura.cli --input parts-data.json --output part.gcode --session settings.aus

El archivo de sesión incluye la configuración del perfil, la impresora y los materiales. Use Archivo-> Exportar sesión ... en Aura para guardar el archivo de sesión con la configuración de corte actual.

# Parámetros

Parámetro Descripción
-m, --modelos (Grupo: partes) Archivos de modelo para cortar, varios nombres de archivo deben estar separados por espacios.
-i, --entrada (Grupo: piezas) Archivo de datos de entrada con información sobre modelos y planchas en formato JSON.
-s, --sesión Necesario. Archivo de sesión con configuración de corte en formato JSON.
-o, --salida Necesario. Archivo de salida.
-c, --centro En el caso de una sola pieza, colóquela en el centro de la placa de construcción.
-f, --force Sobrescriba el archivo de salida existente.
--ayuda Muestra la pantalla de ayuda.
--versión Muestra información sobre la versión.

# Formato de entrada json

Puede especificar una lista de modelos con regiones de disposición y transformaciones mediante el parámetro --input . El esquema JSON para el archivo de entrada se puede encontrar en cli-inputfile.schema.json , que se encuentra en el directorio del programa Aura.

# Ejemplo 1

Cargue dos modelos, gírelos y colóquelos en la placa de construcción

{
    "Models": 
    [
        {
            "Filepath": "spool-axis.stp",
            "Transformation": {
                "Rotation": {
                    "AxisVector": { "X":0, "Y":0, "Z":1 },
                    "AngleRadians": 1.57075
                },
                "Translation": {
                    "X":150, "Y":100
                }
            }
        },
        {
            "Filepath": "spool-axis.stp",
            "Transformation": {
                "Rotation": {
                    "AxisVector": { "X":0, "Y":0, "Z":1 },
                    "AngleRadians": 1.57075
                },
                "Translation": {
                    "X":300, "Y":100
                }
            }
        }
    ]
}

    "Models": 
    [
        {
            "Filepath": "spool-axis.stp",
            "Transformation": {
                "Translation": {
                    "X":100, "Y":100
                }

            "RegionData":

                    "StartHeightMM": 0,
                    "EndheightMM": 20.0,

                        "InnerPlasticWhenFiberCount": 0,
                        "InsetFOuterCount": 3,

                        "FiberInfillType": 2,
                        "IsogridOrientation": 0,

                },
                {
                    "Priority": 2,
                    "StartHeightMM": 0,
                    "EndheightMM": 1000.0,

            ]   
        }
    ]
}

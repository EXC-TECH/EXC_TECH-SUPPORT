My Print Is The Wrong Size

Current Step:
  1. Check scale

Sometimes the intended size of an object can be different between your design software and slicing. If the default units in your modeling software are not millimeters, make sure to change the default units. To do this, navigate to File in MakerBot Print>System Preferences>Units Control. Be sure to save the default model unit you select.

If you open a STL or OBJ in MakerBot Print and it appears to be much too small, click the Scale button once to open the Select a Model Menu and you will be able to rescale your model.

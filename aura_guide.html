1)Iniciando Aura por primera vez #

Después de una instalación exitosa, puede iniciar Aura por primera vez. Simplemente inicie la aplicación en el último paso de instalación o ábrala desde sus aplicaciones.

2)Ventana principal #

Cuando abres Aura, aparece la ventana principal.

Lo primero que puede hacer en Aura es cargar el modelo 3D de una parte. Aura admite los formatos de archivo * .stl, * stp, * .3ds y * .obj.

  • Para cargar el modelo, elija Archivo> Abrir y luego seleccione un modelo en el cuadro de diálogo Abrir archivo. También puede presionar Ctrl + M para llamar al diálogo de apertura del modelo.

Después de cargar el modelo, aparece en una placa de construcción. Ahora puedes guardar un proyecto. Proyecto es la combinación de configuraciones de sesión (perfil + impresora + materiales), modelos en una placa de construcción y sus configuraciones de disposición. El proyecto tiene formato * .auproj.

  • Para guardar el proyecto, elija Archivo> Guardar proyecto como elemento de menú, especifique el nombre del archivo y presione Guardar. O haga clic en Archivo> Guardar ( Ctrl + S ) para sobrescribir el proyecto actual.
  • Para cargar el proyecto, elija Archivo> Abrir proyecto ( Ctrl + P ). Si el proyecto cargado tiene conflictos con la sesión actual, aparecerá un diálogo de conflicto.

3)Configuraciones #

Aura tiene un sistema de configuración modular. Significa que tiene 4 módulos de configuración: plásticos, fibras, impresoras y perfiles. Estos módulos se combinan en una sesión, que contiene todos los parámetros necesarios para el proceso de corte. La sesión contiene una impresora, uno o varios materiales de plástico y fibra y un perfil.

Para cada elemento, puede configurar Nombre, Versión y Descripción. Los nombres de los elementos deben ser únicos. Tenga en cuenta que el nombre del elemento es importante para comparar la configuración en caso de conflicto.

  • Algunos artículos tienen marca aprobada. La marca probada AP significa que esta configuración está aprobada por Anisoprint. Los elementos aprobados no se pueden modificar, pero no te impide realizar experimentos. Si desea realizar cambios, puede copiar la configuración aprobada. Puede encontrar la marca AP a la derecha del nombre de la configuración en la lista de configuraciones.
  • Puede eliminar cualquier elemento de configuración no aprobado (sin marca AP). Simplemente haga clic en el botón Eliminar. Si está intentando eliminar el elemento que está utilizando la sesión, verá una advertencia.
  • Puede duplicar cualquier elemento de configuración. Simplemente haga clic en un botón duplicado. La duplicación crea una copia completa de su elemento de configuración.
  • Puede exportar cualquier elemento de configuración. Simplemente haga clic en un botón de exportación. Cada tipo de configuración tiene su propio formato de archivo.
  • Puede encontrar información sobre cualquier parámetro haciendo clic en un signo de interrogación a la derecha del nombre del parámetro.
  • Puede agregar un nuevo elemento a la lista. Simplemente haga clic en el botón Agregar y verá un nuevo elemento en una lista. El artículo se rellenará con los valores predeterminados.
  • Puede importar un conjunto de configuraciones. Simplemente haga clic en un botón de importación.
  • Puede exportar todas las configuraciones de su lista. Simplemente haga clic en un botón de exportación.

4)Sesión #

La sesión es el agregador de configuraciones. Debe especificar la impresora, el perfil, los materiales en las extrusoras y elegir la extrusora para cada entidad de impresión. La configuración de la parte que puede ver en la parte inferior del panel de sesión se puede ajustar individualmente para cada impresión. Además, esta configuración no se tiene en cuenta durante la comparación de perfiles y la determinación de conflictos. Entonces, si carga un proyecto desde un archivo, esta configuración siempre se anula sin conflicto.

Después de haber configurado la sesión, puede presionar un botón de generación . Si el botón de generación está desactivado, verifique que haya cargado el modelo para cortar y que sus modelos no tengan intersección entre ellos.

5)Conflictos #

El cuadro de diálogo Conflictos aparece si tiene un conflicto con al menos un elemento de configuración en su proyecto (material, impresora o perfil). Conflicto significa que ya tienes en tu configuración de Aura un elemento con el mismo nombre pero con configuraciones diferentes. Por ejemplo, ya tiene material Plástico llamado "PLA" y multiplicador de flujo = 1. Y desea cargar el proyecto que contiene el material "PLA", y este material tiene multiplicador de flujo = 1.1. Por lo tanto, este es el conflicto y debe resolverlo antes de poder continuar trabajando con Slicer. Puedes hacerlo con el cuadro de diálogo Conflicto.

  • Para anular su configuración local, marque una casilla de verificación para ver la configuración adecuada.
  • Para agregar la configuración del proyecto como una copia, desmarque una casilla de verificación para la configuración adecuada.

6)Informe de error #

Si ha encontrado un error mientras trabajaba con Aura o si tiene alguna sugerencia para mejorarlo, envíenos un informe de error. Puede hacerlo desde el panel INFORME DE ERRORES. Describa un error y adjunte el proyecto que produce este error. La descripción completa del error y el archivo adjunto del proyecto que causa el error nos ayudarán a solucionarlo lo antes posible. Si tiene alguna pregunta o desea adjuntar más archivos, envíela a support@anisoprint.com .

7)Panel de modelo #

Después de cargar un modelo, lo verá en una placa de construcción colocada en el centro de la placa de construcción. Puede seleccionar un modelo haciendo doble clic en él. Haga doble clic en el espacio vacío anulará la selección de un modelo. Al hacer doble clic en un panel de modelo de llamadas de pieza.

  • Para eliminar una parte de una placa de construcción, presione un  botón Eliminar.
  • Para mover un modelo en la superficie de la placa de construcción, simplemente presione un botón de movimiento en el panel del modelo. Después de eso, simplemente puede arrastrar una flecha para desplazar una parte a lo largo de los ejes X e Y. O bien, puede especificar manualmente las compensaciones X e Y en los cuadros de texto correspondientes que se encuentran en la parte inferior de una ventana principal. Después de colocar una pieza según sea necesario, presione el botón Aplicar o Entrar para finalizar el movimiento. O presione Cancelar o Esc para cancelar el movimiento.
  • Para rotar una parte alrededor de su centro, presione un botón de rotación en el panel del modelo. Después de eso, simplemente puede rotar un modelo utilizando un manupulador de objetos. O bien, puede especificar manualmente ángulos en cuadros de texto correspondientes que se encuentran en la parte inferior de una ventana principal. Presione el botón Aplicar o Entrar para finalizar la rotación. Presione Cancelar o Esc para cancelar la rotación. Una vez que haya terminado la rotación, una parte se colocará nuevamente en una placa de construcción.
  • Para cambiar el tamaño de una parte, puede presionar un botón de cambio de tamaño en el panel del modelo. Después de eso, puede ingresar valores de escala en los cuadros de texto correspondientes. Presione Aplicar o Entrar para finalizar el cambio de tamaño o haga clic en Cancelar o Esc para cancelar el cambio de tamaño.
  • Para clonar una parte, puede presionar un botón de clonar en el panel del modelo. Después de eso, se creará una nueva parte y se colocará en el centro de la placa de construcción.
  • Para establecer el esquema de estructura de capas , presione un botón para establecer la estructura de capas.

8)Ajuste de la estructura interna de la pieza #

Es posible que desee tener una estructura de pieza no homogénea. Por ejemplo, es posible que desee reforzar con fibra solo las capas inferior y superior para obtener la máxima rigidez a la flexión con una pequeña cantidad de fibra. O es posible que desee reforzar las capas inferiores más que la superior. En general, si tiene requisitos específicos sobre cómo se debe reforzar la pieza en diferentes niveles, puede lograr el resultado con la configuración de la estructura de capas.

Después de presionar un botón de configuración de estructura de capas en el panel del modelo, la ventana cambia su estado al modo de configuración de estructura de capas.

Puede copiar el esquema de capas de otro modelo. Simplemente presione un botón con el ícono selector y luego haga clic en el modelo desde el que desea aplicar la estructura de capas.

Para observar los detalles de la región y editar la región, simplemente selecciónela en el panel Regiones. Configure sus ajustes según lo necesite. Tenga en cuenta que las regiones con el mismo color tienen las mismas propiedades y están conectadas entre sí. Significa que los cambios realizados en una región afectarán a otras regiones con el mismo color.

  • Para agregar región, haga clic en un nuevo botón de región o haga clic y arrastre en el panel de regiones. Asegúrese de que puede especificar los límites inferior y superior de la región y presione el botón Crear.
  • Para eliminar la región, haga clic en el botón Eliminar. Tenga en cuenta que no puede eliminar la última región. Su esquema debe contener al menos una región.
  • Para vincular la región a otra, haga clic en un botón. Esta operación llena la región seleccionada con la configuración de la región seleccionada.
  • Para desvincular la región, haga clic en un botón. Esta operación no cambia la configuración actual de la región, pero ahora todos los cambios realizados en otras regiones no tendrán efecto en esta región y todos los cambios en esta región no tendrán efecto en las regiones vinculadas anteriormente.
  • Para cambiar entre regiones, use los botones. Estos botones son solo para navegar entre regiones

Una vez que haya terminado de configurar el esquema de disposición, no olvide aplicar los cambios presionando el botón Aplicar .

9)Generación de código #

Después de configurar los modelos y una sesión de impresión como desee, presione el botón Generar para generar geometría y traducir al código de máquina. Durante el proceso, puede cancelar la generación con la pérdida de todo el progreso. Cuando la generación finalice con éxito, aparecerá el tiempo de impresión y la información de consumo de materiales. El consumo de materiales se calcula por longitud y masa. Puede guardar en el código de máquina del archivo o abrir el visor de código presionando el botón Ver código .

10)Visor de código de geometría #

Aura tiene dos visualizadores de proceso de impresión: geometría y exploradores de código. La vista de geometría le ayuda a comprender cómo se imprimirá una parte mientras el explorador de código muestra las trayectorias exactas del cabezal de impresión. El visor de geometría muestra entidades tales como carcasa externa, perímetros de plástico, rellenos micro, sólidos y celulares, soportes, perímetros de fibra, etc. El visor de códigos representa entidades como viajes, movimientos, extrusiones puras, retracciones, movimientos con fibra, etc. En ambos modos, usted puede cambiar entre los modos 3D y 2D, puede observar todas las capas, el rango de capas o una capa. Y puede mostrar u ocultar entidades individuales y cambiar sus colores. En el modo de código también puede mostrar y leer el código G, en futuras versiones de Aura agregaremos la posibilidad de seleccionar líneas de código G y resaltar los bloques de código apropiados en el modelo.
